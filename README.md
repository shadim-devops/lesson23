Zadanie 1 - Analiza tekstu

Program pobiera tekst od użytkownika i wyświetla statystyki:
- liczba znaków (ze spacjami)
- liczba znaków (bez spacji)
- liczba słów
- liczba zdań
- najdłuższe słowo
- najczęściej występujące słowo

Uruchomienie:

python zadanie1/analiza_tekstu.py


Zadanie 2 - Książka adresowa

- wyszukać kontakt
- usunąć kontakt
- edytować kontakt
